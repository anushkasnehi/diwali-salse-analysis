•	To provide recommendations for optimizing sales and inventory strategies.


3. Methodology
This outlines the approach taken to conduct the analysis. Summarize the key steps involved in the project:
•	Data Collection: Gather Diwali sales data.
•	Data Preprocessing: Clean and preprocess the data by handling missing values, duplicates, and inconsistencies.
•	Exploratory Data Analysis (EDA): Use Python libraries like Pandas, Matplotlib, and Seaborn to explore the data through visualizations and summary statistics.
•	Advanced Analysis: Apply statistical techniques for deeper insights or predictions.

4. Key Findings
This summarizes the outcomes of the analysis. Highlight the most significant insights discovered, such as:
•	Age group 26-30 contributed the most to revenue.
